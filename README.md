# 🚀 Modern Tech Portfolio

Um portfólio pessoal de alto impacto, desenvolvido com foco em performance, estética minimalista e experiência do usuário. O projeto apresenta uma interface escura (dark mode), animações fluidas e design responsivo.

## ✨ Destaques do Projeto
- **UI/UX:** Design moderno utilizando a tipografia *Space Grotesk* e *Inter*.
- **Interatividade:** Efeitos de hover com emojis animados e cards dinâmicos.
- **Responsividade:** Totalmente adaptável para dispositivos móveis e desktops.
- **Performance:** CSS puro, sem dependências externas pesadas, garantindo carregamento instantâneo.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica.
- **CSS3:** Custom Properties (Variáveis), Flexbox, Grid e Keyframe Animations.
- **JavaScript:** Manipulação de scroll e efeitos de visibilidade (Intersection Observer).
